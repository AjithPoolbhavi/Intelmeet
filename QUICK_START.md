# IntellMeet Premium - Quick Start Guide

## ⚡ 5-Minute Setup

### 1. Clone Repository
\`\`\`bash
git clone https://github.com/yourusername/intellmeet.git
cd intellmeet
\`\`\`

### 2. Install & Configure
\`\`\`bash
# Copy environment files
cp server/.env.example server/.env
cp client/.env.example client/.env

# Install dependencies
npm install

# Start development servers
npm run dev
\`\`\`

### 3. Access Application
- **Frontend**: http://localhost:5173
- **Backend**: http://localhost:5000
- **API Docs**: http://localhost:5000/api/docs

## 🎯 Key Features

### Meeting Room
1. Click "Start Meeting" on dashboard
2. Share meeting link with participants
3. Participants join via link or meeting code
4. Use control bar for:
   - Mic/Camera toggle
   - Screen sharing
   - Chat & Messages
   - Participants list
   - Reactions & Emojis
   - Recording start/stop

### AI Features
1. **Meeting Summary**: After meeting ends, AI auto-generates summary
2. **Action Items**: View extracted action items
3. **Transcription**: View full meeting transcript
4. **Insights**: Get sentiment analysis and highlights

### Collaboration
1. **Chat**: Real-time messaging during meetings
2. **Whiteboard**: Draw and collaborate visually
3. **Tasks**: Create and assign tasks from meetings
4. **File Sharing**: Share documents and files

### Dashboard Analytics
1. **Stats Card**: See meeting metrics at a glance
2. **Trends**: Visualize participation trends
3. **Team Performance**: Monitor team productivity
4. **Meeting History**: Access past recordings

## 📱 User Workflows

### Starting a Meeting
\`\`\`
Login → Dashboard → "Start Meeting" 
→ Grant permissions → Share link/code 
→ Wait for participants → Meeting begins
\`\`\`

### Joining a Meeting
\`\`\`
Login → "Join by Code" → Enter code 
→ Grant permissions → Waiting room 
→ Host admits → You join
\`\`\`

### Scheduling Meeting
\`\`\`
Dashboard → "Schedule" → Enter details 
→ Select date/time → Add participants 
→ Send invites → Meeting scheduled
\`\`\`

### Recording & Download
\`\`\`
Start meeting → Use recording button 
→ End meeting → Visit dashboard 
→ Find recording → Download/share
\`\`\`

## 🆘 Troubleshooting

### Camera/Mic Issues
1. Check browser permissions
2. Verify device is connected
3. Try different browser
4. Restart application

### Connection Problems
1. Check internet speed
2. Close other applications
3. Try wired connection
4. Check firewall settings

### Audio Echo
1. Move microphone away from speaker
2. Use headphones
3. Check volume levels
4. Enable noise suppression

### Video Quality
1. Check bandwidth (recommend 2.5+ Mbps)
2. Reduce video resolution in settings
3. Close background applications
4. Move closer to WiFi router

## 🚀 Advanced Features

### Breakout Rooms
1. Start meeting
2. Click Participants
3. Select "Create Breakout Rooms"
4. Choose number and distribution
5. Participants auto-assigned or manual
6. Join rooms to observe/help

### Virtual Backgrounds
1. Click Camera button → Settings
2. Select "Virtual Background"
3. Choose background image
4. Adjust blur/replacement
5. Apply for meeting

### Live Captions
1. Click Settings → Live Captions
2. Select language
3. Captions appear in real-time
4. Translatable to other languages

### Recording & Transcription
1. Click Record button
2. Recording saves automatically
3. AI transcription starts
4. View transcript in Meeting Details
5. Download as PDF/SRT
